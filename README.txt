This code shows event as they are happening on the Bitcoin network in real time.
This program is a Java based maven project.
This program can be run in number of ways:

1. If using an IDE such a IntelliJ, simply run the Main class from within the program.

2. The application can also be ran using Maven:
	- Install Maven locally
	- Using the command line, cd into the SecurityAssignment3 subdirectory
	- Run "mvn clean install"
	- The application should run using the following command: 
		mvn exec:java -Dexec.mainClass="org.example.Main"

The application has been tested and verified to work on both Windows (Using java 1.8) and Linux (tested using VM using java 11).
